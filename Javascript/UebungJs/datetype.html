📚 JavaScript Datentypen - Vollständige Referenz

Alle JavaScript-Datentypen mit detaillierten Beispielen und Erklärungen

1️⃣ Primitive Datentypen (7 Typen)

String string

Textdaten - Sequenz von Zeichen

// String-Deklaration let str1 = "Doppelte Anführungszeichen"; let str2 = 'Einfache Anführungszeichen'; let str3 = `Template Literal: ${2 + 2}`; let multiline = `Mehrzeilige Strings mit Template Literals`; // String-Methoden let text = "JavaScript"; text.length; // 10 text.toLowerCase(); // "javascript" text.toUpperCase(); // "JAVASCRIPT" text.charAt(0); // "J" text.substring(0, 4); // "Java" text.slice(-6); // "Script" text.indexOf("Script"); // 4 text.replace("Java", "Type"); // "TypeScript" text.split(""); // ['J','a','v','a'...]

Number number

Zahlen - Ganzzahlen und Gleitkommazahlen (64-bit IEEE 754)

// Number-Deklaration let int = 42; let float = 3.14159; let negative = -100; let exponential = 5e3; // 5000 let hex = 0xFF; // 255 let binary = 0b1010; // 10 let octal = 0o17; // 15 // Spezielle Zahlenwerte let infinity = Infinity; let negInfinity = -Infinity; let notANumber = NaN; // Number-Methoden Number.isInteger(42); // true Number.isNaN(NaN); // true Number.parseFloat("3.14"); // 3.14 Number.parseInt("42"); // 42 (3.14159).toFixed(2); // "3.14" (1234.5).toExponential(); // "1.2345e+3"

Boolean boolean

Wahrheitswerte - true oder false

// Boolean-Deklaration let isTrue = true; let isFalse = false; // Boolean aus Vergleichen let isGreater = 5 > 3; // true let isEqual = 10 === 10; // true let isNotEqual = 5 !== 3; // true // Truthy und Falsy Werte Boolean(1); // true Boolean(0); // false Boolean(""); // false Boolean("text"); // true Boolean(null); // false Boolean(undefined); // false Boolean(NaN); // false Boolean([]); // true Boolean({}); // true

Undefined undefined

Variable deklariert, aber nicht initialisiert

// Undefined-Beispiele let x; console.log(x); // undefined function test() { // keine return-Anweisung } console.log(test()); // undefined let obj = { a: 1 }; console.log(obj.b); // undefined typeof undefined; // "undefined"

Null null

Absichtliche Abwesenheit eines Wertes

// Null-Beispiele let empty = null; console.log(empty); // null // Unterschied zu undefined let notDefined; // undefined (nicht zugewiesen) let intentionallyEmpty = null; // null (absichtlich leer) typeof null; // "object" (bekannter Bug!) empty === null; // true empty == undefined; // true (lose Gleichheit) empty === undefined; // false (strikte Gleichheit)

Symbol symbol

Eindeutige und unveränderliche Werte (ES6)

// Symbol-Erstellung let sym1 = Symbol(); let sym2 = Symbol("Beschreibung"); let sym3 = Symbol("Beschreibung"); sym2 === sym3; // false (jedes Symbol ist einzigartig) // Symbol als Object Key let id = Symbol("id"); let user = { name: "Max", [id]: 12345 }; // Symbol.for() - Globale Symbole let gs1 = Symbol.for("app.id"); let gs2 = Symbol.for("app.id"); gs1 === gs2; // true

BigInt bigint

Große Ganzzahlen über Number.MAX_SAFE_INTEGER hinaus (ES2020)

// BigInt-Erstellung let big1 = 9007199254740991n; let big2 = BigInt("9007199254740991"); let big3 = BigInt(100); // BigInt-Operationen let sum = 100n + 200n; // 300n let product = 10n * 20n; // 200n let power = 2n ** 100n; // sehr große Zahl // Größenvergleich Number.MAX_SAFE_INTEGER; // 9007199254740991 9007199254740991n > Number.MAX_SAFE_INTEGER; // true // WICHTIG: Kann nicht mit Number gemischt werden // 10n + 5; // TypeError!

2️⃣ Objekt-Typen (Reference Types)

Object object

Sammlungen von Key-Value-Paaren

// Object-Erstellung let obj1 = {}; // Objekt-Literal let obj2 = new Object(); // Object-Konstruktor let obj3 = Object.create(null); // Objekt ohne Prototype // Objekt mit Eigenschaften let person = { name: "Max Mustermann", age: 30, isActive: true, address: { street: "Hauptstr. 1", city: "Berlin" }, greet: function() { return `Hallo, ich bin ${this.name}`; } }; // Zugriff auf Eigenschaften person.name; // "Max Mustermann" person["age"]; // 30 // Object-Methoden Object.keys(person); // ['name', 'age', 'isActive', ...] Object.values(person); // Alle Werte Object.entries(person); // Key-Value-Paare Object.assign({}, person); // Shallow Copy

Array object (array)

Geordnete Listen von Werten

// Array-Erstellung let arr1 = []; let arr2 = new Array(); let numbers = [1, 2, 3, 4, 5]; let mixed = [1, "text", true, null, {}]; // Array-Methoden numbers.push(6); // Element hinzufügen numbers.pop(); // Letztes Element entfernen numbers.shift(); // Erstes Element entfernen numbers.unshift(0); // Am Anfang einfügen numbers.length; // Anzahl Elemente // Array-Iterationsmethoden numbers.forEach(n => console.log(n)); numbers.map(n => n * 2); // [2,4,6,8,10] numbers.filter(n => n > 3); // [4,5] numbers.reduce((a, b) => a + b); // Summe numbers.find(n => n > 3); // 4 numbers.some(n => n > 3); // true numbers.every(n => n > 0); // true

Function function

Wiederverwendbare Code-Blöcke

// Function-Deklaration function greet(name) { return `Hallo ${name}!`; } // Function-Expression const greet2 = function(name) { return `Hi ${name}!`; }; // Arrow Function (ES6) const greet3 = (name) => `Hey ${name}!`; // IIFE (Immediately Invoked Function Expression) (function() { console.log("Sofort ausgeführt"); })(); // Funktionen sind "First-Class Objects" const functions = [greet, greet2, greet3]; const obj = { method: greet };

Date object (Date)

Datum und Zeit

// Date-Erstellung let now = new Date(); let specific = new Date("2026-03-04"); let timestamp = new Date(1678886400000); let parts = new Date(2026, 2, 4, 12, 30, 0); // Date-Methoden now.getFullYear(); // Jahr now.getMonth(); // Monat (0-11) now.getDate(); // Tag now.getDay(); // Wochentag (0-6) now.getHours(); // Stunden now.getMinutes(); // Minuten now.getTime(); // Timestamp (ms seit 1970) now.toLocaleDateString("de-DE"); now.toLocaleTimeString("de-DE"); now.toISOString(); // ISO-Format

RegExp object (RegExp)

Reguläre Ausdrücke für Muster-Matching

// RegExp-Erstellung let regex1 = /abc/; let regex2 = new RegExp("abc"); let regex3 = /abc/gi; // g=global, i=case-insensitive // RegExp-Methoden let text = "JavaScript ist großartig"; /script/i.test(text); // true text.match(/\w+/g); // ['JavaScript', 'ist', 'großartig'] text.replace(/JavaScript/, "JS"); text.search(/ist/); // 11 (Position) // Häufige Muster let email = /^[^\s@]+@[^\s@]+\.[^\s@]+$/; let phone = /^\d{3}-\d{3}-\d{4}$/; let url = /^https?:\/\/.+/;

3️⃣ Weitere wichtige Typen

Map

let map = new Map(); map.set("key1", "value1"); map.set("key2", 42); map.get("key1"); map.has("key2"); map.delete("key1"); map.size;

Set

let set = new Set(); set.add(1); set.add(2); set.add(1); // Duplikat set.has(2); // true set.size; // 2 set.delete(1);

WeakMap

let wm = new WeakMap(); let obj = {}; wm.set(obj, "value"); wm.get(obj); wm.has(obj); wm.delete(obj); // Nur Objects als Keys!

WeakSet

let ws = new WeakSet(); let obj = { id: 1 }; ws.add(obj); ws.has(obj); ws.delete(obj); // Nur Objects!

Promise

let promise = new Promise((resolve, reject) => { setTimeout(() => { resolve("Erfolg!"); }, 1000); }); promise.then(result => { console.log(result); }).catch(error => { console.error(error); });
