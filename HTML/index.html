Struktur-Tags

<header> - Kopfbereich

Definiert einen Kopfbereich für ein Dokument oder eine Sektion.

<header>
  <h1>Willkommen</h1>
  <nav>Navigationsmenü</nav>
</header>

<nav> - Navigation

Kennzeichnet einen Navigationsbereich mit Links.

<nav>
  <ul>
    <li><a href="#home">Home</a></li>
    <li><a href="#about">Über uns</a></li>
  </ul>
</nav>

<main> - Hauptinhalt

Enthält den Hauptinhalt der Seite. Sollte nur einmal pro Seite verwendet werden.

<main>
  <article>...</article>
</main>

<footer> - Fußbereich

Definiert einen Fußbereich für ein Dokument oder eine Sektion.

<footer>
  <p>&copy; 2026 Meine Website</p>
</footer>

Medien

<img> - Bild

Fügt ein Bild ein.

<img src="bild.jpg" alt="Beschreibung" width="300" height="200">
Wichtig: Das alt-Attribut ist wichtig für Barrierefreiheit!

<video> - Video

<video width="320" height="240" controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Ihr Browser unterstützt das Video-Tag nicht.
</video>

<audio> - Audio

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Ihr Browser unterstützt das Audio-Tag nicht.
</audio>
